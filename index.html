IoT-based Home Automation System

This model is built on home automation techniques which allows us to automate our homes with various sensors and provides consumers with access to web applications that allow them to remotely control their home equipments. The project is designed and built in C++ using the Arduino IDE. A smartphone or computer can be used to access and manage the appliances. The ESP8266 microcontroller serves as the core hub or controller. It can be set up to connect with a variety of sensors and devices in the home, including an infrared sensor, gas sensor and other appliances.
